## 📱 About

Reitti is a freemium Android app that shows Helsinki region public transport in real time. Built by someone who actually commutes daily — tired of waiting in the cold not knowing where the bus is.

- 🇫🇮 **Suomi + English** — täysin käännetty
- 🔒 **No data collection** — emme kerää henkilötietoja
- 📡 **Official HSL open data** — MQTT + GTFS-RT streams
- 🆓 **Freemium** — perustoiminnot ilmaisia, premium lisäominaisuuksille

---

## ✨ Features

| Feature | Details |
|---------|---------|
| 📍 **Real-Time Tracking** | All 5 transport modes on one map — 3-second position updates |
| ⏱️ **Arrival Times** | Direct from HSL GTFS-RT feeds, color-coded by urgency |
| 🗺️ **Route Visualization** | Interactive route maps with all stops highlighted |
| 🔔 **Smart Notifications** | Get alerted when your bus approaches — works in background |
| ⭐ **Favorites** | Save stops and lines for one-tap access from the home screen |
| 🔍 **Fast Search** | Find any stop or line in seconds by name or number |
| 📤 **Share Locations** | Send a stop or vehicle location to a friend |
| 👁️ **Vehicle Follow** | Track a specific vehicle and get notified at every stop |
| 🎨 **Map Themes** | Light, dark, and custom map styles |

### 🚦 Transport Modes
🚌 Buses &nbsp; 🚊 Trams &nbsp; 🚇 Metro &nbsp; ⛴️ Ferries &nbsp; 🚆 Commuter Trains

## 🔧 Tech Stack

- **Platform:** Android (API 24+)
- **Real-time data:** HSL MQTT feed → vehicle positions every 3 seconds
- **Schedule data:** HSL GTFS-RT (trip updates, vehicle positions, alerts)
- **Maps:** Interactive map with custom vehicle overlays
- **Languages:** Finnish 🇫🇮 · English 🇬🇧
